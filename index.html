This app helps you try out queries as specified in the DOM Testing Library docs.


  1. First, write some HTML markup in the HTML editor area.
  2. Find a query you're interested in.
  3. Write the query in the query editor area and see if any elements match in the output area.

An example query: 

queries.getByText(container, "username")


queries and container are special keywords. queries is an import from testing library itself, which has all the queries, and container is the output area div. These are always mandatory.
